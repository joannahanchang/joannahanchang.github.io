Week 3:

  • I was introduced to JavaScript, which is not related to the Java programming language and it is case sensitive.
  • I learned that I would be learning about vanilla JavaScript before delving into JavaScript-based technologies: React, React Native, Express.js, Node.js, etc. in this bootcamp.
  • While there are more data types in JavaScript, the lesson focused on these: number, string, boolean, null, and undefined.
  • I learned about variable declaration (using let, const, and var), assignment, and initialization.
  • I learned to use let for variables where the values could not be reassigned, and use const for variables where the values will not be reassigned.
  • Pre-ES6, JavaScript only had one variable declaration keyword called var. Let and const were introduced to correct some issues with the way var works.
  • I learned about defining, calling, and returning functions, as well as about using parameters and arguments.
  • If statements were introduced: if, if...else, if...else if...else
  • If statements check for whether a condition is truthy or falsy. A non-Boolean value that would be true if converted to Boolean is considered truthy; likewise, a non-Boolean value that would be false if converted to Boolean is falsy.
  • Another conditional statement introduced was the switch statement, which is used to perform different actions based on different conditions. This is a good solution for long, nested if/else statements.
  • I learned two types of comparison operators, equality and relational. Equality operators refer to strict/loose equality/inequality. Relational operators include greater/less than, and greater/less than or equal to.
  • Common logical operators in JavaScript include &&, ||, and !. && and || are binary, while ! is unary. && returns the last truthy value if both operands are truthy; if either operand is falsy, then && returns the first falsy value. || returns the first truthy value; if neither operand is truthy, then it will return the last falsy value. ! is a logical not and !! is the double not operator.
  • Additional operators include addition assignment operators (+=), subtraction assignment operator (-=), increment operator (++), and decrement operator (--). Addition and subtraction assignment operators are binary, while increment and decrement operators are unary.
  • Loops are a way to repeat a block of code, and I learned about while loops and do...while loops this week. While loops evaluate a condition that you set, and it will continue to loop as long as the condition evaluates as true. With the while loop, the condition is evaluated first, so if the condition is false in the beginning, the code inside the block never executes. With the do...while loop, the loop block executes once before evaluating the condition - so even if the condition is false, the code inside the block will always execute at least once.
  • I learned 7 different types of array methods: shift, unshift, join, includes, and indexOf. Shift removes an item from the beginning of an array, while unshift adds one more items to the beginning of an array. Join takes all items in an array anda returns a string containing those items. Includes is used to check if an item exists inside an array, and will return true or false. IndexOf also checks if an item is in an array, but returns the index of the item if it exists in the array.
  • I learned about the math global object and math.random and math.floor to help generate random numbers. Math.random()*6 for example can generate a number between 0-6, but not including 6. Math.floor removes all digits to the right of the decimal point from a number. If we want to generate a random number between 1-6 rather than 0-5, we can add 1 to the result: const randomInteger = Math.floor(Math.random()*6)+1.
  • Code Challenges: There were 2 code challenges this week, which can be found in my CodePen: Variables and Functions Practiceand Switch.
  • Project: I built a Number Guessing Game using JavaScript, including functions, variables, arrays, operators, if statements, do...while loops, etc.

Week 4:

  • I learned the difference bewteen for looks and while loops: for loops are like while loops, but with built-in support for an incrementing iterator. It is in this format: for(initialization; condition; final expression){.
  • For...of loops works with arrays and other iterables (data structures that contain lists of items that can be iterated over). It can be expressed as: for (const/let variableName of arrayName) {.
  • Loops can be controlled in the form of break or continue. Break statement will immediately exit a loop, and continue statement will immediately skip to the next iteration of the loop.
  • I learned more array methods this week, including concat, sort, reverse, slice, and splice (which can abe used for inserting, adding to, or removing items from an array at any point, not only the beginning or the end.
  • String methods include indexOf, includes, slice, toLowerCase, toUpperCase, and split.
  • I learned about DOM, the Document Object Model providing an interface between JavaScript and HTML. According to W3 school, it is an application programming interface (API) for valid HTML and well-formed XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated. A tree graph example would be html at the top (root node/parent node of head and body) -> head and body in the middle (child nodes of html). Head and body in this case are also sibling nodes.
  • As mentioned in the previous point, node relationship properties could include parentNode, firstChild, lastChild, nextSibling, previousSibling, and childNodes. We can use to reference nodes in relation to each other.
  • 4 ways to locate nodes are: traversing the DOM, getElementsByTagName, getElementById, querySelector.
  • 4 ways to create and adding nodes are: createElement, createTextNode, insertBefore, and appendChild.
  • 3 ways to remove nodes are: remove, removeChild, and firstElementChild.
  • Nodes can also be cloned using cloneNode and cloneNode(true).
  • A few mouse events include onclick, onmousedown, onmouseup, onmouseover, and onmouseout.
  • We can also set time events using setTimeout, clearTimeout, setInterval, and clearInterval.
  • Events can be added or removed using JavaScript via addEventListener and removeEventListener. One advantage of addEventListener over other ways of adding event handlers is that we can use it to add more than one handler function to the same event, on the same element.
  • Code Challenge: There was 1 code challengeq this week, which can be found in my CodePen: Arrays Practice.
  • Project: I built a Matching Game using DOM manipulation, adding event listeners, etc.
